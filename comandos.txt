Add: É o comando para adicionar um arquivo no controle de versão
Status: É o comando para saber quais arquivos foram incluídos no controle de versão ou não.
Commit: São as versões dos códigos
Pull: É o comando que traz as atualizações do servidor para a máquina local.
Push: É o comando para enviar as versões de códigos para nuvem
Clone: Copiar um repositório
Pull request: É uma solicitação de aprovação do meu commit
